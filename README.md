# Aplicación de Reserva de Videojuegos – Versión 2

## Descripción
Segunda versión del proyecto PlayPoint, una aplicación ficticia diseñada para gestionar la reserva de videojuegos.  
En esta versión se han añadido los archivos de estilos `style.css` y `reserva.css` para mejorar la apariencia visual de las interfaces JavaFX.

La aplicación está planteada para permitir en un futuro:
- Consultar un catálogo de videojuegos.
- Gestionar reservas activas.
- Administrar perfiles de usuario.

## Tecnologías implicadas
- Figma
- Java
- JavaFX
- CSS
- Maven
- IntelliJ IDEA
