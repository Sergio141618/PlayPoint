# Aplicación de Reserva de Videojuegos – Versión 1

## Descripción
Proyecto de diseño y documentación de una aplicación ficticia para la reserva de videojuegos.  
Esta versión contiene la estructura inicial del proyecto y la creación de la clase `Usuario`.

La aplicación está planteada para permitir en un futuro:
- Consultar un catálogo de videojuegos.
- Gestionar reservas.
- Administrar perfiles de usuario.

## Tecnologías implicadas
- Figma
- Java
- JavaFX  
- Maven  
- IntelliJ IDEA  
- Git / GitHub
